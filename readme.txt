(App Ecommerce Android Studio)

Đây là hướng dẫn cài đặt và build ứng dụng:
1. Cài đặt Anroid Studio. (Android Studio 13.0)
2. Cấu hình SDK (ở đây em sử dụng SDK 33)
3. Build project để cập nhật tệp tin cũng như import thư viện
4. Chọn thiết bị để chạy máy ảo
5. Nhấn nút run để chạy chương trình.

(Admin Panel)

1. Cài đặt python 3.9
2. Download và giải nén source
3. Tại vị trí python\app\ecommerce chạy lệnh pip install -r requirements.txt
4. Tiếp tục chạy lệnh python manage.py runserver sẽ lấy được đường dẫn localhost.
5. copy đường dẫn vào trình duyệt chạy.
